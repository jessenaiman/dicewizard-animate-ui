Use these concise guides to stay consistent when building and styling components in this project. They reflect project-specific patterns and defaults.

<Callout>
  - Prefer named imports from `motion/react`.
  - Default Motion spring: `stiffness: 150`, `damping: 22`.
  - Components expose `data-slot` attributes for styling hooks.
</Callout>

See individual pages in the sidebar for details.
