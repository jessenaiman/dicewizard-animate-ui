---
title: Lucide Icons
description: Using `lucide-react` for consistent iconography.
---

## Import

```tsx
import { PlusIcon, XIcon } from 'lucide-react';
```

## Usage

```tsx
<PlusIcon className="size-4" aria-hidden />
<XIcon className="size-5 text-muted-foreground" aria-label="Close" />
```

Icons inherit `currentColor`; size via Tailwind (e.g. `size-4`).
