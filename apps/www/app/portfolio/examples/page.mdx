# Examples

<Orbit size={20} style={{ verticalAlign: 'text-bottom' }} /> Import single‑page demos (Astro → MDX) here. Each example can be a dedicated `.mdx` page in this folder.

## Converting Astro → MDX

1. Copy the HTML/MD content from the Astro page into a new `*.mdx` file here.
2. Replace Astro components with equivalent React or Animate UI components.
3. Move images into `/portfolio/assets/examples/` and reference with absolute paths.
